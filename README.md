AJAX-powered entrepreneurial mindset assessment with admin dashboard, results scoring, CSV/PDF export, and email notifications.

== Description ==

Custom Assessment provides a shortcode-driven assessment flow with:

- Step-by-step questions and progress tracking
- Automatic score calculation and profile summaries
- Admin dashboard for submissions, questions, and categories
- Export options (CSV and PDF) for completed submissions
- Email delivery of completed results to respondents

Use shortcode: `[custom_assessment]`

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress Plugins screen.
3. Add the `[custom_assessment]` shortcode to any page or post.
4. Open the admin menu to manage questions, categories, and submissions.

== Frequently Asked Questions ==

= How do I display the assessment? =

Place `[custom_assessment]` on a page or post.

= Where do I view submissions? =

Go to the plugin's admin dashboard in WordPress admin.
